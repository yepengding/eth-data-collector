# Ethereum Data Collector

Simple Ethereum data collector.

- [x] Gas data

## Quickstart

1. Install dependencies

```shell
npm install
```

2. Duplicate `config.example.js` as `config.js` and customize it

3. Run project

```shell
npm run start
```

## API

- Default port: 3000
- Get current gas price: `/gas/current/price`
